## node版本
```
v16.17.0
```

### 安装前端
```shell
$ git clone https://gitee.com/namegen/agen_music.git
$ cd agen_music
$ npm install
```
### 前端运行
```js
npm run serve
```

### 安装后端
node.js 
##### 文档官网:https://binaryify.github.io/NeteaseCloudMusicApi/#/
```shell
$ git clone https://github.com/Binaryify/NeteaseCloudMusicApi.git
$ cd NeteaseCloudMusicApi
$ npm install
```
